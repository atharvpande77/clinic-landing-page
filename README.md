# Attreya Wellness & Healing Clinic

This repository contains the source code for the Attreya Wellness & Healing Clinic web application.

## Overview

The application is built using React and TypeScript, powered by Vite for fast development and building.

## Getting Started

### Prerequisites

Ensure you have Node.js and npm installed on your machine.

### Installation

Install the project dependencies:

```bash
npm install
```


Start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173` (by default).

### Build

To build the application for production:

```bash
npm run build
```

## Technologies

- **Frontend**: React, TypeScript, Vite
- **AI Integration**: Google GenAI SDK

## Project Structure

- `src/` (or root in this case since `App.tsx` is in root): Application source
- `components/`: UI Components
- `server/`: Backend/Server-side code
- `public/`: Static resources
